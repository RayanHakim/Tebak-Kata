# 🎮 Tebak Kata

Game Tebak Kata berbasis website yang dibuat menggunakan HTML, CSS, dan JavaScript. Pemain akan diberikan sebuah kategori dan harus menebak kata yang dipilih secara acak.

## 🌐 Demo

https://rayanhakim.github.io/Tebak-Kata/

## ✨ Fitur

- 🎲 Kata dipilih secara acak.
- 📚 Banyak kategori.
- 📱 Responsive untuk HP dan Desktop.
- 🔤 Sistem tebakan huruf.
- ❤️ Nyawa pemain.
- 🏆 Sistem menang dan kalah.
- 🔄 Main lagi tanpa refresh halaman.

## 🚀 Menjalankan Project

Clone repository:

```bash
git clone https://github.com/RayanHakim/Tebak-Kata.git
```

Masuk ke folder:

```bash
cd Tebak-Kata
```

Buka:

```
index.html
```

atau gunakan Live Server di VS Code.
